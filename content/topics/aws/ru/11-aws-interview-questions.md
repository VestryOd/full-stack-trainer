# AWS Interview Questions (Fullstack / Senior)

---

# 1. Что такое AWS?

AWS (Amazon Web Services) — облачная платформа Amazon, предоставляющая сервисы для вычислений, хранения данных, сетей, безопасности, очередей сообщений и баз данных.

---

# 2. Что такое Cloud Computing?

Модель предоставления вычислительных ресурсов через интернет по требованию.

---

# 3. Что такое Region?

Географический регион AWS.

Пример:

```txt
eu-central-1
us-east-1
```

---

# 4. Что такое Availability Zone?

Изолированный датацентр внутри региона AWS.

---

# 5. Зачем нужны Availability Zones?

Для отказоустойчивости.

Если один датацентр недоступен, приложение продолжает работать в других AZ.

---

# 14. Что такое CloudFront?

CDN AWS.

---

# 15. Что такое CDN?

Сеть серверов для ускоренной доставки контента.

# 70. Самый популярный Senior вопрос

Когда выбрать Lambda, а когда ECS/Fargate?

Ответ:

Lambda подходит для коротких event-driven задач и нерегулярной нагрузки. ECS/Fargate лучше подходит для долгоживущих API, контейнерных приложений, WebSocket соединений и сервисов с постоянной нагрузкой.

---

# 71. Самый сильный архитектурный ответ

Как выбрать AWS сервис?

Ответ:

Нельзя выбирать сервис по принципу "всегда использую Lambda" или "всегда использую ECS". Выбор зависит от характера нагрузки, требований к latency, стоимости, отказоустойчивости, времени выполнения и операционной сложности системы.
