# Read Replicas

Самый популярный подход.

---

```txt

 ↓

 ↓

Replica 2

Replica 3
```

```txt

```

---

# Частый вопрос

Когда использовать Read Replicas?

Ответ:

Когда система имеет значительно больше операций чтения, чем записи.

---

# Частый вопрос

Когда использовать Sharding?

Ответ:

Когда вертикальное масштабирование и реплики уже не решают проблему нагрузки.

---

# Interview Answer

Масштабирование БД обычно начинается с Read Replicas. Если этого недостаточно, используются шардирование, CQRS или разделение данных между сервисами.
